# рҹ“– About the Project

**Cyber Shield** is a real-time cybersecurity application that monitors network traffic, detects suspicious activities, generates security alerts, and automatically blocks malicious IP addresses. The system helps administrators monitor network security through an interactive dashboard with live analytics and reports.

This project was developed as a **Cybersecurity Major Project** using Python and Streamlit.
